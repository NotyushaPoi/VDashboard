# 🎬 VDashboard - 主播展示平台

一个现代化的主播展示平台，展示你喜爱的主播、她们的歌单和追番列表。

## ✨ 核心功能

- 🎪 **自动轮播展示** - 主播宣传图自动播放，点击进入详情页
- 🎵 **歌单管理** - 展示歌曲，点击[歌词]按钮查看歌词，支持一键复制
- 📺 **追番列表** - 管理追番列表，标记观看状态
- 🌙 **Dark Mode** - 完整的深色模式支持
- 📱 **响应式设计** - 完美适配手机、平板、桌面
- ⚡ **高性能** - 静态生成 (SSG)

## 🚀 快速开始

```bash
cd vdashboard

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 打开 http://localhost:3000
```

# Images Directory Structure

此目录存放项目所需的所有静态图片文件。

## 文件夹说明

### `/video`
存放主页轮播视频的封面图片
- 文件命名: `video{N}-cover.jpg`
- 用途: 首页视频轮播卡片的缩略图
- 示例: `video1-cover.jpg`, `video2-cover.jpg`, ...

### `/streamer`
存放主播相关的图片资源
- 主播头像、banner、背景图等
- 用途: 主播详情页和个人页面展示

### `/avatar`
存放头像相关图片（如果需要单独管理）
- 用途: 用户头像、主播小头像等

## 添加图片

1. 在对应文件夹中添加图片文件
2. 在相应的YAML文件中更新路径引用:
   - 视频封面: `vdashboard/data/videos.yaml`
   - 主播信息: `vdashboard/data/streamers/{slug}.yaml`
   - 其他资源: 相应的数据文件

## 图片规范

### 推荐尺寸

| 图片类型 | 尺寸 | 比例 | 用途 |
| --- | --- | --- | --- |
| 用户/主播头像 | 200×200px | 1:1 | 个人资料 |
| 主播列表头像 | 120×120px | 1:1 | 网格卡片 |
| 视频封面 | 480×270px | 16:9 | 轮播卡片 |
| 主播Banner | 1200×400px | 3:1 | 详情页顶部 |
| 页面背景 | 1920×1080px | 16:9 | 全屏背景 |

### 图片格式建议

- **头像**: JPG (质量90), PNG (支持透明)
- **Banner/背景**: JPG (质量80-90), WebP (备选)
- **视频封面**: JPG (质量85), WebP (备选)
- 建议使用工具压缩优化以提高加载速度

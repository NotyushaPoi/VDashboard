#### 功能演示
1. **轮播组件**
   - 自动每 5 秒切换一张
   - 点击 ◄ / ► 手动切换
   - 点击下方圆点快速跳转
   - 鼠标悬停暂停自动播放

2. **卡片功能**
   - 显示头像首字母（占位）
   - 显示粉丝数（加载中...）
   - 显示歌单/番剧数量
   - 点击卡片或[查看详情]跳转到子页面

3. **Dark Mode**
   - 点击导航栏右上角切换
   - 所有颜色自动调整

## 🎨 Dark Mode 效果

### Light Mode（默认）
```
背景: 白色          → 背景: 浅灰色
文字: 黑色          → 文字: 深灰色
卡片: 白色          → 卡片: 浅灰色
按钮: 彩色          → 按钮: 彩色（更亮）
```

### Dark Mode
```
背景: 深灰色        → 背景: 黑色
文字: 白色          → 文字: 浅色
卡片: 深灰色        → 卡片: 更深的灰色
按钮: 彩色          → 按钮: 彩色（适配暗色）
```

**切换方式**: 点击导航栏右上角的太阳(☀️)/月亮(🌙)图标

## 🔄 数据流展示

### 点击轮播图流程
```
用户点击轮播图
    ↓
跳转到 /streamer/1 (希罗Kirara)
    ↓
页面加载子页面内容
    ↓
显示主播信息和标签页
```

### 点击歌词按钮流程
```
用户点击 [歌词]
    ↓
模态框弹出
    ↓
显示歌词内容
    ↓
用户可复制歌词或关闭
    ↓
点击 X 或关闭按钮
    ↓
模态框消失
```

### 点击复制按钮流程
```
用户点击 [复制]
    ↓
歌曲名复制到剪贴板
    ↓
按钮变绿色，显示"已复制 ✓"
    ↓
2秒后自动恢复原样
```

## 🎯 完整使用场景

### 场景 1：用户第一次访问
1. 进入主页
2. 看到轮播展示 6 位主播
3. 向下滚动查看主播卡片
4. 点击感兴趣的主播卡片
5. 进入主播详情页

### 场景 2：查看歌曲和歌词
1. 在子页面看到歌单标签
2. 点击某首歌的[歌词]按钮
3. 在弹出的模态框中查看歌词
4. 可选：复制歌词内容
5. 关闭模态框，继续浏览

### 场景 3：追番管理
1. 点击[📺 追番]标签页
2. 查看追番列表和观看进度
3. 点击番剧名打开外链
4. 可选：复制番剧名

### 场景 4：切换深色模式
1. 在任何页面点击导航栏右上角的主题按钮
2. 页面颜色立即变为深色
3. 刷新页面，深色模式保持
4. 再次点击按钮恢复亮色模式
